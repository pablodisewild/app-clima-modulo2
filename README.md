# App Clima - Módulo 2
### Estudiante: Pablo Ignacio Muñoz García

## Descripción
MVP de aplicación meteorológica desarrollada para el Módulo 2 de Front-End. Incluye un listado dinámico de 10 localidades, navegación a vista de detalle con pronóstico semanal y diseño responsivo utilizando Bootstrap 5 local. Enfocado en HTML5 semántico y JS Vanilla.
